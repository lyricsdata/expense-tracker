  ヒートマップの読み方

  これは GitHubの草（contribution graph）と同じ形式で、1年間の「日ごとの支出額」を一目で見るためのものです。

          ← 週(1月)              週(12月) →
  Mon  □ ■ □ □ ▓ □ ...
  Tue  □ □ ▓ □ □ ■ ...
  Wed  □ ▓ □ █ □ □ ...
  ...
  Sun  □ □ □ □ ▓ □ ...

  - マス1個 = 1日
  - 横軸（左→右）= 週：左端が1月初め、右端が12月末。横に進むと時間が進みます。
  - 縦軸（上→下）= 曜日：上がMon（月）、下がSun（日）。同じ縦の列が「同じ週の月〜日」です。
  - 色の濃さ = その日の支出額：
    - 薄い黄色 = 支出が少ない（または0）
    - 濃いオレンジ〜赤 = 支出が多い（YlOrRd = Yellow→Orange→Red）
  - マスにマウスを乗せると、その日の日付と金額（2025-08-21 / SGD 120.00）が出ます。

  何がわかるか

  - 赤いマスが縦に並んでいる週 → その週は出費が多かった
  - 特定の曜日（行）が赤くなりがち → 例えば週末に使いすぎ、など曜日のクセが見える
  - 赤い1マスがポツンとある → その日に大きな買い物をした（家賃やrentなど）
